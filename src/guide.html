Для валидации формы необходимо добавить соответствующие класс и аттрибут:

<form class="needs-validation" novalidate>

все элементы формы должны быть внуртри тега form, кнопка должна быть submit

<button type="submit" class="btn bubbly-button bubbly-button-green">send</button>

Общая структура поля

<label class="form-control-label with-ic">
    <input type="text" class="form-control" placeholder="Пароль" required>
    <div class="form-control-img"><svg class="ic-svg"><use xlink:href="images/icons/sprite.svg#ic-lock"/></svg></div>
    <span class="form-control-title">Пароль</span>
</label>

Для поля с иконкой, необходимо добавить класс with-ic

<label class="form-control-label with-ic">

А обязательным полям необходимо добавить аттрибут required:

<input type="text" class="form-control" placeholder="Фамилия Имя Отчество" required>

Когда поле неактивное, добавляется аттрибут и\или класс disabled

<input type="text" class="form-control disabled" placeholder="Фамилия Имя Отчество" disabled>

Когда поле валидное, то добавляется класс is-valid

<input type="text" class="form-control is-valid" placeholder="Фамилия Имя Отчество" required>

Когда поле невалидное, то добавляется класс is-invalid

<input type="text" class="form-control is-invalid" placeholder="Фамилия Имя Отчество" required>

Классы is-valid, is-invalid добавляются автом-ки при проверке формы

Зеленая кнопка

<button type="submit" class="btn bubbly-button bubbly-button-green">send</button>

Розовая кнопка

<button type="submit" class="btn bubbly-button bubbly-button-pink">send</button>
